# dotfiles

managed by [chezmoi](https://www.chezmoi.io/)

## set up

1. [install chezmoi](https://www.chezmoi.io/install/)

2. init chezmoi

   ```sh
   chezmoi init https://github.com/Kohey821/dotfiles.git
   ```

3. apply chezmoi

   ```sh
   chezmoi apply
   ```

4. run set up script

   - Ubuntu/macOS

     ```sh
     ./set-up.sh
     ```

   - Windows

     ```ps1
     .\set-up.ps1
     ```
